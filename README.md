# Sistema de Busca de Produtos - MegaStore

## Descrição
Este projeto foi desenvolvido com o objetivo de melhorar a busca de produtos em um sistema de e-commerce. 

A ideia é permitir que os produtos sejam encontrados de forma mais rápida e organizada, utilizando a linguagem Rust.

## Funcionalidades
- Cadastro de produtos
- Busca de produtos pelo nome
- Listagem de todos os produtos
- Busca de produtos por categoria

## Tecnologias utilizadas
- Rust
- Estrutura de dados HashMap

## Como executar o projeto

No terminal, dentro da pasta do projeto, execute:

cargo run
